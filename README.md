# PrepBuddy – Interview Preparation & Job Portal

PrepBuddy is a full-stack platform for interview preparation, experience sharing, coding practice, forums, and job hunting.  
Technologies: React (frontend), Node.js + Express (backend), MySQL (database), Material UI + Framer Motion (UI).

---

## 1. Features

- User authentication (JWT)
- Auto-fetching company logos
- Company listing + details page
- Interview experience submission
- Company-wise experience display
- Forum system:
  - Create posts
  - Add replies
  - Threaded replies (parent-based)
  - Upvotes (one per user)
  - Accepted answers
- Leaderboard scoring
- Coding practice categories
- Graduate Programs (Ireland) with search filters
- Profile page showing:
  - user details
  - their experiences
  - their forum posts

---

## 2. Tech Stack

Frontend: React, Material UI, Framer Motion  
Backend: Node.js, Express  
Database: MySQL  
Auth: JWT  
Deployment: Railway (DB), Netlify (frontend), Render (backend)
